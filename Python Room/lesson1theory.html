Lesson1 : Basic Input/Output Operations




Introduction



Input and output (I/O) operations allow a program to interact with users. Input operations take data from users, while output operations display data to users. These operations are fundamental in programming for data processing, user interaction, and debugging.In Python, we use:
input() to take user input.
print() to display output.




1.The print() Function (Output Operation)

The print() function is used to display data on the screen.

Syntax:

print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)


objects: The values to print.

sep: Separator between values (default: space ' ').

end: String appended at the end (default: newline '\n').

Basic Print Example:

print("Hello, World!")
Output:
Hello, World!
