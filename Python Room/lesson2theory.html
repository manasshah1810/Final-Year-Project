Python Variables and Data Types

Introduction

Variables and data types are fundamental concepts in programming. A variable stores data in memory, and data types define the kind of data that can be stored.

Declaring Variables

In Python, you can create a variable by assigning a value to a name:

x = 10  # Integer variable
y = "Hello, World!"  # String variable
z = 3.14  # Float variable

Basic Data Types

  • int: Whole numbers (e.g., 5, -3, 100)
  • float: Decimal numbers (e.g., 3.14, -2.5)
  • str: Text values (e.g., "Python", "Hello")
  • bool: Boolean values (True or False)
  • list: Ordered, mutable collection (e.g., [1, 2, 3])
  • tuple: Ordered, immutable collection (e.g., (1, 2, 3))
  • dict: Key-value pairs (e.g., {"name": "Alice", "age": 25})

Examples

age = 25  # Integer
gpa = 3.85  # Float
name = "Alice"  # String
is_student = True  # Boolean
grades = [90, 85, 88]  # List
student_info = {"name": "Alice", "age": 25}  # Dictionary

Type Checking

Use the type() function to check the type of a variable:

print(type(age))  # Output: <class 'int'>
print(type(name))  # Output: <class 'str'>
print(type(gpa))  # Output: <class 'float'>
